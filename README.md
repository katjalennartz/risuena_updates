# risuena_updates
Updatefile um meine Plugins aktuell zu halten. Nötig damit updates durchgeführt werden können. Aktualisiert wenn nötig, Templates, Datenbank, Settings etc. Muss bei den meisten meiner Plugins vorhanden sein,

## Anleitung
Ihr müsst die Datei einfach nur hochladen, es ist keine Installation nötig. Achtet darauf sie ins richtige Verzeichnis zu laden.  
inc/plugins/risuena_updates/risuena_updatefile.php
